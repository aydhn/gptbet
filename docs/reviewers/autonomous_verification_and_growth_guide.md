# Autonomous Verification and Growth Guide

## Verification Windows
Reviewers must understand that the autopilot uses temporal windows to assess health. A "clean" short window is a prerequisite for early growth.

## Evaluating Growth Decisions
When the autopilot recommends growth, it provides an evidence packet. Review the citations and rationale before approving transitions that require manual oversight (e.g., Level 2 to Level 3).
