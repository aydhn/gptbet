# Transparency Anomaly Adjudication Guide

Reviewers must adjudicate anomalies escalated from the Witness Mesh.

## Process
1. Review the `TransparencyAnomalyRecord`.
2. Inspect the linked `ChallengeRecord` and `WitnessStatementRecord`s.
3. Determine the outcome (e.g., `LOCAL_MIRROR_ISSUE` or `INTEGRITY_BREAK_CONFIRMED`).
4. Record the decision and rationale in an `AnomalyAdjudicationRecord`.
