# Federated Decision Taxonomy

A reference for all federated governance decisions and artifacts.

- **EscalationCase**: When a plane requests help from a higher level.
- **PlaneBudget**: Risk budget allocated to a plane.
- **EmergencyOverride**: A forceful action taken by a high precedence plane.
