# Continuity Verification Taxonomy

## Observatory Federation
A bounded collection of observatory nodes producing a synthesized view.
-   Families: bounded, degraded, archive_assisted, etc.
-   Statuses: verified, caveated, review_only, gapped, blocked, overclaimed.

## Scheduler Proof Lane
A verifiable channel carrying scheduling proofs.
-   Families: planetary_coverage, archive_restore, etc.

## Audit Pulse Council
An evaluator of audit evidence.
-   Families: follow_the_sun, no_safe_visibility, etc.

## Continuity Evidence Exchange
A bounded mechanism for matching evidence requests with listings.
-   Families: scheduler_truth, archive_lineage, etc.
