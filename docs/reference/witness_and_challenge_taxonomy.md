# Witness and Challenge Taxonomy

- **WitnessFamily**: local, mirror, federated, etc.
- **WitnessCapability**: verify_checkpoint, verify_inclusion, issue_challenge, etc.
- **StatementType**: checkpoint_verified, mirror_divergence_confirmed, etc.
- **ConsensusType**: unanimous_confirmed, majority_confirmed, split_observation, contested, etc.
- **ChallengeStatus**: opened, awaiting_response, resolved, escalated, etc.
- **AnomalyOutcome**: benign_explained, integrity_break_confirmed, quarantine_required, etc.
