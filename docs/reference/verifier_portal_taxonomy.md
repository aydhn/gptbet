# Verifier Portal Taxonomy

This reference covers the structured terms and profiles used throughout the verifier portal layer.

## Audience Profiles
- `public_viewer`: Standard public access.
- `registered_verifier`: Known verifiers with challenge capabilities.
- `trusted_external_verifier`: Vetted verifiers with deep proof visibility.
- `external_auditor`: Complete systemic access for oversight.
- `quarantine_reviewer`: Dedicated reviewers for flagged items.
- `internal_preview_viewer`: System operators testing views.
