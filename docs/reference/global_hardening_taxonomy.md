# Global Hardening Taxonomy

- **Mesh / Path / Lag / Edge:** Descriptors for global node connectivity.
- **Seam / Reachability / Window / Handoff:** Descriptors for planetary operator coverage.
- **Continuity / Residue / Checkpoint / Phase:** Descriptors for disaster drill execution.
- **Approval / Fallback / Decision Window:** Descriptors for wide-area recovery governance.
- **Budget / Breach / Warning:** Resilience safety thresholds.
