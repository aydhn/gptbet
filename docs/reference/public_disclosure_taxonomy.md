# Public Disclosure Taxonomy

## Profiles
- `public_minimal`: Shallow proofs, no challenge intake
- `public_verifier`: Moderate proofs, challenge intake
- `external_auditor`: Deep proofs, broad families allowed
- `trusted_exchange_partner`: Deep proofs for partners

## Families
- `policy_bundle_disclosure`
- `transparency_checkpoint_disclosure`
- `readiness_summary_disclosure`
- `anomaly_summary_disclosure`
