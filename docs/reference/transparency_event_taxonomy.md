# Transparency Event Taxonomy

Logs are divided into families:
- `policy_transparency_log`
- `governance_decision_log`
- `approval_transparency_log`
- `signer_trust_log`
- `override_transparency_log`
- `activation_rollback_log`
- `federated_import_log`
- `cross_plane_event_log`

Events include:
- `signed_policy_bundle_published`
- `policy_bundle_promoted`
- `critical_decision_proof_created`
- `emergency_override_issued`
- `family_freeze_applied`
- `signer_trust_changed`
...
