# Cohort Decision Taxonomy

- `PROGRESS_COHORT`: Increase activation level.
- `HOLD_COHORT`: Maintain current level, await more data.
- `PAUSE_COHORT`: Temporarily halt processing/growth due to warnings.
- `SHRINK_COHORT_SCOPE`: Reduce the segment definition.
- `ROLLBACK_COHORT`: Revert to the stable reference.
