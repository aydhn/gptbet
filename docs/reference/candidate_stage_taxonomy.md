# Candidate Stage Taxonomy

1. **Integrity Validation**: Schema checks, valid payloads, no forbidden targets.
2. **Safety Validation**: Blast radius estimation, scope safety, ops mode compatibility.
3. **Simulation Validation**: Fair baseline vs variant comparison, materiality bounds met.
4. **Quality Gates Validation**: Required smoke, regression, and scenario suites.
5. **Review Readiness Validation**: Secondary review requirements met, unresolved disputes cleared.
6. **Release Readiness Validation**: Canary-prep checks, hold/kill evaluations.
