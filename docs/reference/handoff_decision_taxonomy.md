# Handoff Decision Taxonomy

- `APPROVE_HANDOFF`: Candidate is fully ready for bridge transfer.
- `HOLD_FOR_MORE_EVIDENCE`: Awaiting additional data or approvals.
- `REJECT_HANDOFF`: Candidate failed council evaluation.
- `KILL_CANDIDATE_BEFORE_HANDOFF`: Candidate is stale, superseded, or fundamentally unsafe.
