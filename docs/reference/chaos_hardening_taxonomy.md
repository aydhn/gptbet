# Chaos Hardening Taxonomy

- **Chaos Probes**: Controlled scenarios to test fault behavior.
- **Fault Injection**: Specific points where faults (timeout, exception, delay) are simulated.
- **Degradation Rehearsal**: Simulating complex failure states to observe the recovery path.
- **Recovery Honesty**: Validating that recovery does not overclaim success.
- **Failure Visibility**: Ensuring faults are surfaced, not silently swallowed.
