# Terminal Lifecycle Hardening Taxonomy

- **ClosureBundleRecord**: Captures a verified set of terminal states.
- **DeprecationMapRecord**: Represents the state transitions of legacy surfaces.
- **MaintenanceModeRecord**: Defines the boundaries and constraints of a frozen but active surface.
- **LongHorizonStewardshipPackRecord**: Details the ownership, escalation, and cadence rules for a maintained surface.
