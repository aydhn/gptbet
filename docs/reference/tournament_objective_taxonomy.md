# Objective Taxonomy

Typical metrics used to form the objective vector:
- `decision_quality_delta` (Maximize)
- `dispute_burden_delta` (Minimize)
- `review_queue_pressure_delta` (Minimize)
- `support_strength` (Maximize)
- `estimated_release_risk` (Minimize)
- `scope_breadth` (Minimize)
