# Federation Hubs, Baselines, and Ecosystems Guide

This guide explains how to operate the Federation Ecosystem:
- Use `run-federation-ecosystem-pass` to project registry currentness, simulate hub admission/routing, and evaluate ecosystem visibility.
- Review output artifacts (e.g., `federation_ecosystem_summary.json`) for hub queue pressure and federation health.
- Remember: `discoverability != authorization`. A bounded exchange route requires explicit local admission.
