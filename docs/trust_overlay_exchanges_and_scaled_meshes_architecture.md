# Trust Overlay Exchanges and Scaled Meshes Architecture

Phase 81 introduces Trust Exchange Scale discipline to the federated ecosystem.
It prevents unbounded authority escalation when meshes grow and overlays are exchanged.

## Core Concepts
1. **Overlay Exchanges**: Shared bounded interpretations, not runtime authorities. Caveats are preserved.
2. **Scaled Meshes**: Hub meshes partitioned by scope and pressure. High pressure degrades route class, not scope bounds.
3. **Benchmark Signal Ecologies**: Evaluates corroboration, provenance, and freshness. Suppresses stale signals.
4. **Governance Baselines**: Sovereign resilience norms. Drift reduces projection strength via controller caps.

`python -m sports_signal_bot.main trust-exchange-scale run-trust-exchange-scale-pass`
